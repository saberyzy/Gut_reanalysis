This folder includes all scripts for the upstream pipeline for the Gut paper. For each script, two versions (notebook and python) are presented

Steps:

1. Run files_org.py
2. put run_count.sh and cellranger_count.py in the same path, run run_count.sh
3. Run cellranger_aggr_prep.py
4. Run aggr.sh
5. Run preprocessing.py

The parameters in each scripts need to be adjusted depend on the actual cases
